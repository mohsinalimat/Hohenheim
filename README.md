# Introduction

## What's this?

`Hohenheim` is an elegant image picker for iOS.

![preview.jpg](./Assets/preview.jpg)

## Requirements

* iOS 10.0+
* Xcode 9 with Swift 4

## Installation

#### CocoaPods

```ruby
pod 'Hohenheim'
```
